## Why Mizzle?

<CardGrid stagger>
	<Card title="Developer Velocity" icon="rocket">
		Minimize boilerplate when defining tables and entities with a Drizzle-like experience.
	</Card>
	<Card title="100% Type Safe" icon="approve-check">
		Ensure type safety for queries, updates, and schema definitions out of the box.
	</Card>
	<Card title="Single-Table Design" icon="setting">
		First-class support for advanced patterns like key prefixing and static keys.
	</Card>
	<Card title="Fluent API" icon="open-book">
		Familiar `db.select()`, `db.update()`, and `db.delete()` API for intuitive data access.
